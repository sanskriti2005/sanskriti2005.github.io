asteroid-tracker

This Python script fetches information on near-Earth asteroids within a specified date range using NASA's Near Earth Object Web Service (NeoWs). NeoWs is a RESTful web service for near earth Asteroid information. My CLI application takes the start_date and end_date as arguments and outputs the name, Asteroid ID as well as NEO Reference ID of the asteroid(s) within the specified date range.
